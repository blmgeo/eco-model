EcoModel

A light-weight JavaScript library for environmental data analysis and ecosystem modeling.

Diversity

The diversity module provides calculations for various diversity indices. A diversity index is a numeric measurement of unique species that exist within a ecological community.

Predator-Prey Models

The Lotka-Volterra module provides calculations for predator-prey dynamics. A predator-prey relationship is characterized by the dynamic balance between the two populations over time.

Steady State Systems

The Steady State module can model real-world steady state processes. A system is considered steady state when it remains unchanged after a dynamic process has acted upon it.

Non-Steady State Systems

The Non-Steady State module models dynamic systems. A system is non-steady state if it is changed by dynamic processes over time.

Population Growth Functions

The Sigmoid module currently supports S-shaped growth models. Sigmoids are characterized by slow initial growth, then rapid growth that evens out.
